### Adding Component 

To add a new MCU/MPU, perform the following steps:

1. Click [ + Add MCU/MPU ].
2. Select the target platform.
3. Choose the MCU/MPU vendor.
4. Select the desired target board.
5. Click [ Add selected ] to confirm.

After the MCU/MPU is added, it appears in the MCU/MPU Sidebar Panel, where it becomes available for further configuration.\
\
Currently, the platform supports MCU/MPU models for Zephyr target. The target can be selected in the Add MCU/MPU dialog, where the user can also specify the vendor and choose the target board of interest.
