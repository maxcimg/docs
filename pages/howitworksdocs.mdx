All code generated through Embedd can be completed with comprehensive Doxygen-compatible documentation. This documentation provides detailed insights into the code structure, function descriptions, and usage examples.

To generate documentation for your component:

1. Navigate to your Component Model
2. Click the "Generate Documentation" button in the top menu
3. A new folder containing the documentation will be created in your downloads

The generated documentation includes:

- Detailed API reference
- Function descriptions and parameters
- Usage examples and code snippets
- Component configuration details

All documentation is generated in HTML format and can be easily integrated into your existing documentation system.
